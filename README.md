# CubesDOTS
Unity demo using DOTS (Entity Component System, C# Jobs and Burst compiler).


## Installation
- Install Unity 2019.1.14f1
- Install the following packages using Package Manager:
    - Entities.
    - Jobs.
    - Burst.
    - Unity Renderer.
    - Unity DOTS.
    - DOTS for Android.
    - DOTS for macOS.
    - DOTS for Windows.


## Playing Demo
- Open and play 'Demo.unity' scene.
